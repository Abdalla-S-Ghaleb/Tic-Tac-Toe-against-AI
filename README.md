# Tic-Tac-Toe AI

A Tic-Tac-Toe game featuring an AI opponent powered by the **Minimax algorithm**.

## Overview

This project was built to explore the fundamentals of **Artificial Intelligence** and **adversarial search**. The AI analyzes all possible game states using the Minimax algorithm and selects the optimal move at every turn, resulting in an unbeatable opponent when both players play optimally.

## Features

* Play against an AI opponent
* AI powered by the Minimax algorithm
* Optimal decision-making for every game state
* Clean implementation for learning AI search algorithms

## Technologies

* Language: Python
* Algorithm: Minimax

## What I Learned

* Implementing the Minimax algorithm from scratch
* Modeling game states and state transitions
* Recursive decision-making and backtracking
* Applying AI search techniques to solve deterministic games
